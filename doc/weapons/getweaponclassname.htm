getweaponclassname(<weapon name>)


Module 
            Weapons


Summary


            
Returns the classname for the given weapon

Example


            
weapon_type = getweaponclassname("mosin_nagant_sniper");

Minimum Number of arguments: 1


                

  • 1 : <weapon name> The name of the weapon
