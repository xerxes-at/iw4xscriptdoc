Custom Events

Module: IW4x
MP Only

Say Event:

level waittill("say", string, player);



Args:

  • 1 : <string> The chat message.
  • 2 : <client> The client entity.


Done Event:

response waittill("done", success, data);


See httpget for an example.


Args:

  • 1 : <success> The success status.
  • 2 : <data> The returned data.
