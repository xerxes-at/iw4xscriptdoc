Obituary( <victim>, <attacker>, <weapon>, <Means of death> )

Module: HUD
MP Only

Summary:

Create an obituary for a character



Example:

Obituary( self, attacker, sWeapon, sMeansOfDeath );



Required Args:

  • 1 : <victim> The victim entity
  • 2 : <attacker> The attacker entity
  • 3 : <weapon> The weapon name
  • 4 : <weapon> The means of death as a string
