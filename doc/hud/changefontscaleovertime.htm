ChangeFontScaleOverTime( <time> )

Module: HUD
SP Only

Summary:

Set a hud element to transition in fontScale over time. Normally setting the fontScale of an element causes an immediate visual change. However, if the fontScale gets set within <time> after calling ChangeFontScaleOverTime, then the hud element will transition to the new fontScale over the remaining <time>.



Call this on:

Hud Element



Example:

level.introstring1 ChangeFontScaleOverTime( 1.2 );  level.introstring1.fontScale = 0.3;  // This will transition the fontScale from whatever it was before to the new value of 0.3 over 1.2 seconds.



Required Args:

  • 1 : <time> The time to scale the element font in seconds
