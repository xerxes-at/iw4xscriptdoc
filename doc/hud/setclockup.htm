SetClockUp( <time>, <full time>, <material>, <height>, <width> )

Module: HUD

Summary:

Set a clock HUD element to count up over a time period.



Call this on:

Hud Clock Element



Example:

level.bombstopwatch SetClockUp( 5, 60, "hudStopwatch", 64, 64 );



Required Args:

  • 1 : <time> The new timer time in seconds to count up
  • 2 : <full time> The time in seconds represented by a full cycle of the hand, for instance a stopwatch would be 60s.
  • 3 : <material> The material for the hud clock


Optional Args

  • 1 : <width> The width of the material.
  • 2 : <height> The height of the material.
