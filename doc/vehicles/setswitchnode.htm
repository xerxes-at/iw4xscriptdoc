SetSwitchNode( <source node> <dest node> )

Module: Vehicles
SP Only

Summary:

Sets a switch node for this vehicle.



Call this on:

a vehicle



Example:

tank2 SetSwitchNode( tank2snode1, tank2snode2 );



Required Args:

  • 1 : <source node> The switch source node
  • 2 : <dest node> The switch destination node.
