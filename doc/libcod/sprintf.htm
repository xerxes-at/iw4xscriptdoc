sprintf( <format>, <additional_arguments> )

Module: libcod

Summary:

Composes a string with the same text that would be printed if format was used on printf.
See http://www.cplusplus.com/reference/cstdio/sprintf/.
Exception: no buffer/stringpointer is required for sprintf, returns the string to stack instead.



Example:

 text = sprintf("Hello %s!", "COD");



Required Args:

  • 1 : <format> string that contains a format string that follows the same specifications as format in printf.
  • 2 : <additional_arguments> Depending on the format string, the function may expect a sequence of additional arguments, each containing a value to be used to replace a format specifier in the format string.
    There should be at least as many of these arguments as the number of values specified in the format specifiers. Additional arguments are ignored by the function.
