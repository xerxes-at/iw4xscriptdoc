load_languages( <path_and_filename>, <force_reload> )

Module: libcod

Summary:

Scans a file for a predetermined format of language items, very similar to CoD2 localized strings (.str files) and can be stored as any file, any extension.
Call add_languages with the path+filename of said file. Second input is a force-reload flag which in the future might reduce loading times by a few milliseconds.
For now, not-reloading is bugged and the force-reload flag should always be set to 1.



Example:

 load_languages(getcvar("fs_homepath") + "/" + getcvar("fs_game") + "/translations/JH_b3.txt", 1);



Required Args:

  • 1 : <path_and_filename> path to language file
  • 2 : <force_reload> force reloading language file
