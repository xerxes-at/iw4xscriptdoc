MayMoveToPoint( <Position> )

Module: AI
SP Only

Summary:

Check whether the actor can move to a given point.



Call this on:

<actor> An actor



Example:

         if ( !self MayMoveToPoint( endPoint ) )...



Required Args:

  • 1 : <position> (point) The position that the actor may be able to move to.


Optional Args

  • 1 : <check drop> checks if the point would cause the AI to drop. Defaults to true.
