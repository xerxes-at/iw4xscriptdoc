FindReacquireProximatePath( <ignore suppression> )

Module: AI
SP Only

Summary:

Finds a path to a point that can see the enemy without leaving goal. Call ReacquireMove to use the path.



Call this on:

<actor> An actor



Optional Args

  • 1 : <ignore suppression> Whether to ignore suppression when looking for a path. True makes a success more likely but may result in a more dangerous path.
