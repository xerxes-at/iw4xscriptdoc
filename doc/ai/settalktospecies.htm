SetTalkToSpecies( <species>, ... )

Module: AI
SP Only

Summary:

Sets the species this AI can share enemy information with.



Call this on:

<actor> An actor



Example:

SetTalkToSpecies( "dog", "human



Optional Args

  • 1 : <species> list of species, if not specified, makes this AI talk with no one; use "all" to talk with all species
