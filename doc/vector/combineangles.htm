CombineAngles( <angles a>, <angles b> )

Module: Vector

Summary:

Returns angles b reoriented by angles a.



Example:

crateAnglesInWorld = CombineAngles( boatAngles, crateAnglesInBoat );



Required Args:

  • 1 : <angles a> Angles representing the orientation in which to place <angles b>
  • 2 : <angles b> Angles which will be reoriented
