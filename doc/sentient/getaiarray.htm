GetAIArray( <team>, <team>, ... )

Module: Sentient
SP Only

Summary:

Returns an array of the human AI



Example:

 aiarray = GetAIArray( "axis", "neutral" );



Required Args:

  • 1 : <team> a team name, either 'axis', 'allies', or 'neutral'


Optional Args

  • 1 : <team> any number of additional team names may be added, either 'axis', 'allies', or 'neutral'
